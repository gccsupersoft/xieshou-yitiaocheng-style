# 写手一条城风格 Skill

> 把抽象社会议题，拆成普通人的生活成本、身份焦虑、评价体系和关系筹码。

这是一个面向 AI Agent 的表达风格 Skill，用于生成接近 B 站 UP 主「写手一条城」公开视频中常见的短视频标题、口播开头、脚本大纲和改稿建议。

它不是本人，也不代表本人观点。它蒸馏的是公开内容中可观察的表达方法：社会现实剖片、县城观察、绰号化人群切片、戏谑反讽、狠话金句和关系结构拆解。

## 效果示例

用户：

```text
使用 $xieshou-yitiaocheng-style，帮我把这个选题改成写手一条城风格：

选题：为什么年轻人越来越讨厌公司团建？
目标：3-5 分钟 B 站口播视频
要求：给 10 个标题、一个开头、一个脚本大纲。
```

Skill 输出方向：

```text
标题：

1. 公司团建，是打工人最便宜的情绪税
2. 你以为团建是放松，其实是老板在低成本测试服从性
3. 《职场浮世绘》之「最怕周五下午通知团建」
4. 团建不是福利，是一种带薪表演
5. 为什么年轻人越来越讨厌团建？

口播开头：

公司团建这东西，最魔幻的地方不是它不好玩。

是所有人都知道它不好玩，但所有人还得装作自己很感动。

你以为老板让你去露营，是想让你亲近自然？不是。是想看看你离开工位以后，还能不能继续保持一种低成本的亢奋。

脚本结构：

1. 点破误区：团建不是休息，是职场关系考试。
2. 拆三类人：积极表演型、被迫营业型、精神离职型。
3. 拆利益结构：公司花小钱买气氛，员工花周末交情绪税。
4. 给反转：年轻人不是不合群，是不想把私人时间也交给组织评价体系。
5. 金句收尾：真正好的团队，不靠团建证明团结。
```

## 安装

发布到 GitHub 后，可以用 skills CLI 安装：

```bash
npx skills add gccsupersoft/xieshou-yitiaocheng-style
```

也可以本地安装：

```bash
npx skills add ./xieshou-yitiaocheng-style
```

安装后，在支持 Skills 的 Agent 中这样触发：

```text
使用 $xieshou-yitiaocheng-style，帮我把这个选题改成一条城风格。
```

或者直接说：

```text
帮我按写手一条城风格写 10 个标题。
```

## 适合做什么

| 任务 | 输出 |
|---|---|
| 短视频标题 | 县城系列、公共议题审判、伪学术教程、反常识狠话 |
| 口播开头 | 先给判断，再给生活场景，再做关系拆解 |
| 脚本大纲 | 人群命名、场景定位、利益结构、反转、金句收尾 |
| 原稿改写 | 把“太正、太散、太讲道理”的稿子改得更短促、更戏谑 |
| 选题生成 | 从县城、婚恋、消费、编制、教育、反内卷等场景里组合选题 |
| 降低冒犯度 | 保留锋利表达，把人身攻击改成机制观察 |

## 蒸馏了什么

这个 Skill 没有把写手一条城简化成“县城梗机器”。

调研播放量 Top 10 后可以看到，他的爆款不只来自「县城瑜伽裤」这类高辨识度标签，也来自教育批判、性别议题、消费换算、普通男生压力和反内卷表达。

因此本 Skill 抽取的是更底层的表达系统：

| 层级 | 内容 |
|---|---|
| 社会现实剖片 | 把宏大议题拆成普通人的生活成本 |
| 县城显微镜 | 用县城、熟人社会、相亲、编制、消费符号承载观察 |
| 绰号化人群 | 用“县城好大哥”“县城王漫妮”“县城小布尔乔亚”等标签制造画面 |
| 伪学术拆解 | 用“技术流分析”“生存指南”“顶级心法”包装世俗经验 |
| 关系经济学 | 拆谁在拿捏谁，谁用什么筹码换什么东西 |
| 狠话金句 | 用短句、反转句、可截图句压住结尾 |

## 使用边界

这个 Skill 是表达风格参考，不是身份模拟。

- 不声称自己是写手一条城。
- 不替本人发表观点。
- 不复刻大段原文。
- 不输出可被误认为本人新作品的内容。
- 涉及性别、地域、职业、年龄等群体标签时，把攻击性转成机制观察。
- 涉及事实、热点和数据时，需要先核验最新资料。
